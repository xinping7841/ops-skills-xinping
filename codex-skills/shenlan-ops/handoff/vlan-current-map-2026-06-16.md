1. VLAN11/12/13/16/17/18 是什么业务？如果还在用，建议补 description；如果不用，后续可以规划清理或保留但标注“预留”。

3. XGE1/0/28 当前 brief 显示 trunk/PVID1/TO-Huawei5130，但旧备份里曾是 access VLAN30；需要单独查接口当前配置。
4. VLAN50/70/90/110/201 有部分 `undo dhcp select server` 或 DOWN，需要确认是否静态地址、外部 DHCP、还是历史遗留。
5. OpenWrt `network.@route[16-18]` 对 VLAN16/17/18 缺少 UCI gateway 字段，但内核路由当前存在；后续做 mwan3 前建议规范化这些路由。
