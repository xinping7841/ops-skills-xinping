### Hunyuan3D Runtime Notes

- Start/restart with `systemctl --user restart hunyuan3d.service` or `bash /opt/hunyuan3d/start_hunyuan3d_gradio.sh` for foreground debugging.
- `custom_rasterizer_kernel` was compiled in `/opt/hunyuan3d/Hunyuan3D-2.1/hy3dpaint/custom_rasterizer` and installed editable into `/opt/hunyuan3d/hunyuan3d21`.
- `hy3dpaint/custom_rasterizer/__init__.py` exports `rasterize` and `interpolate` from `.custom_rasterizer.render`.
- The start script exports `LD_LIBRARY_PATH` with the PyTorch library path and `/usr/local/cuda/lib64`, plus `PYTHONPATH` entries for the project, `hy3dpaint`, and `custom_rasterizer`.
- The start script exports `PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True`.
- Texture defaults are low-VRAM tuned for the 24GB 4090D: `HUNYUAN3D_TEX_MAX_VIEWS=4` and `HUNYUAN3D_TEX_RESOLUTION=512`. Raise these only after validating GPU headroom.
- `/generation_all` smoke test passed on 2026-06-25 with 5 shape steps, octree resolution 128, and generated `textured_mesh.glb` in about 27 seconds.
